Lab 8 - Anon Functions

Challenge

The challenge of this lab is to experiment and familiarize ourselves with anon functions and callback functions. We were to define an array, and then take the elements from that array and put it through functions that would do various mathematical conversions on them. We did this by using callback functions and anon functions.

Problems

The main problem was getting used to the syntax, and figuring out the difference between anon and callback functions. I see now that anon functions are just functions without a name, and callback functions are functions that can be used as parameters for another function.

Reflection

This assignment was pretty fun to work through, I like problem solving and was good practice for me as someone who hasn't coded in a while. I think the things that you can do with functions are pretty cool, and it is interesting to see Java be so lenient and accept functions as parameters for other functions.
